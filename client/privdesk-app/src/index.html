连接组件被安全软件拦截

PrivDesk 的连接功能被系统或安全软件拦截了。这是远程工具的常见误报(并非病毒),请按下面任一方法放行后重试:

Windows 智能应用控制
设置 → 隐私和安全性 → Windows 安全中心 → 应用和浏览器控制 → 智能应用控制 → 选择"关闭"
360 / 火绒 / 电脑管家
打开杀毒软件 → 找到拦截记录 / 信任区 → 把 PrivDesk 添加为"信任"或"允许运行"
Windows Defender
Windows 安全中心 → 病毒和威胁防护 → 管理设置 → 排除项 → 添加 PrivDesk 安装目录

放行后,回到 PrivDesk 重新点击"连接"即可。
